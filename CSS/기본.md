### :ocean: CSS 적용방식

- 전체 선택자

  ```html
  <style>
      *{
          margin:0;
      }
  </style>
  ```

  "*" 을 통해서 해당 웹문서 전체에 적용

  

- 타입 선택자

  ```html
  <style>
      p{
          font-style: italic;
      }
  </style>
  ```

  태그 이름을 적어줌으로 해당 태그에 CSS 적용

  

- 클래스 선택자

  ```html
  <style>
      .bg_color{
          background-color: #ffff;
      }
  </style>
  
  ...
  
  <div>
      <h1 class="bg_color">
          제목
      </h1>
  </div>
  ```

  적용하고픈 CSS 을 클래스로만든다. ( "." 필수) 그리고 해당 태그에  클래스 적용



- id  선택자

  클래스 선택와 비슷하다 

  그런데 다른점은 "." 이 아닌 "#" 을 쓰고 id 라는 것

  

- 그룹 선택자

  ```html
  //여러선택자를 합치는것
  h1 ,p{
  	text-align: center;
  }
  ```
