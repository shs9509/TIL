:first_quarter_moon: **웹 애플리케이션? 웹사이트?**

- **웹 애플리케이션** : 웹브라우저와 웹기술을 사용하여 사용자와 대화하는 대화식으로 인터넷을 사용하는 컴퓨터 프로그램

  ex) 요즘 댓글도 달수있고 검색도 할수있는 블로그나 사이트들

  

- **웹 사이트** : 웹페이지의 모음 

  ex) 그냥 볼수만 있는 뉴스페이지나 블로그
